
Zeb11 Starter OS
================
This is a lightweight web-based simulation of an OS called Zeb11. It includes:

- Taskbar with app launch buttons
- Control Center
- Wallpaper background
- Zeb GX browser stub
- Zeb Store stub
- Settings stub

To get started:
1. Open index.html in a browser or live server (VS Code)
2. Future additions: .NET interop, iPad touchscreen, VPN/proxy, storage drivers

Phase 1 is UI. Future phases include:
- C#/Rust modules
- VPN/Proxy engine
- Epic Games Launcher compatibility
